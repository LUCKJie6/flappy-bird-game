# Flappy Bird 小游戏

一个使用原生 HTML、CSS 和 JavaScript 编写的网页版 Flappy Bird 小游戏。

项目不依赖后端服务，也不需要安装任何第三方库，下载后直接双击 `index.html` 就可以在浏览器中游玩。

## 游戏介绍

玩家需要控制小鸟不断向上飞，躲避从右向左移动的管道。

小鸟会受到重力影响持续下落，按空格键或点击游戏画面可以让小鸟向上飞。成功穿过一组管道可以获得 1 分；如果小鸟碰到管道或地面，游戏结束。

## 游戏特点

- 使用最基础的前端技术实现：HTML、CSS、JavaScript
- 不使用 React、Vue、Canvas 库或第三方游戏引擎
- 支持键盘空格键和鼠标点击操作
- 支持实时分数显示
- 支持游戏结束后重新开始
- 提供三种难度：简单、普通、困难
- 页面采用简洁的像素风 / 卡通风样式

## 难度说明

- 简单：小鸟下落更慢，管道移动更慢，管道间隙更大，适合新手练习。
- 普通：速度和管道间隙适中，适合熟悉玩法后挑战。
- 困难：管道更快、间隙更小，更接近经典 Flappy Bird 的挑战感。

## 操作方式

- 按 `空格键`：让小鸟向上飞
- 鼠标点击游戏画面：让小鸟向上飞
- 点击 `重新开始`：重新开始一局游戏

## 如何运行

1. 下载或克隆本项目。
2. 找到项目中的 `index.html` 文件。
3. 直接双击 `index.html`，用浏览器打开即可开始游戏。

也可以使用 Git 克隆：

```bash
git clone https://github.com/LUCKJie6/flappy-bird-game.git
cd flappy-bird-game
```

然后双击 `index.html` 运行。

## 技术实现

游戏主要逻辑都写在 `script.js` 中，包括：

- 重力效果：每一帧增加小鸟的下落速度
- 跳跃控制：按空格或点击时给小鸟一个向上的速度
- 管道生成：定时创建上下两根管道
- 管道移动：每一帧让管道从右向左移动
- 碰撞检测：判断小鸟是否碰到管道或地面
- 计分逻辑：小鸟成功穿过一组管道后加 1 分
- 难度切换：根据选择的难度调整重力、跳跃力度、管道速度和管道间隙

## 项目预览
